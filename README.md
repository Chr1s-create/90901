# How to use

 1. Clone repository to your local machine
 2. Open Folder
 3. Install dependencies
 ```bash
 npm install
 ```
 4. Navigate to .../90901/packages/client and run
 ```bash
 npm run build
 ```
 5. Navigate back to root folder .../90901
 6. run app
 ```bash
 npm start
 ```
 7.Open App on localhost:3000
